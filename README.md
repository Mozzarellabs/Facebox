# Facebox

DIY portable photo booth. 
Uses a Raspberry Pi, a camera and a thermal printer. 

##Basic behavior 

1. Plug in power 
2. LED switches on when ready 
3. Press button 
4. LED blinks - get ready ! 
5. Picture is taken (LED switches off) 
6. Picture prints 

¯\_(ツ)_/¯

Press the LED longer to swith power off.
